 Herramienta de Renombrado Masivo

            
Convierte nombres genéricos de imágenes a nombres descriptivos para asignación automática.

            
            

                
 Paso 1: Descargar guía CSV

                
Descarga este archivo CSV que contiene todos tus productos. Úsalo como referencia para renombrar imágenes.

                
                
                

                    
 Paso 2: Ejemplos de renombrado

                    

                        

                            
ANTES: DSC_1234.jpg

                            
DESPUÉS: CORBAG10K.jpg

                        

                        

                            
ANTES: IMG_4567.jpg

                            
DESPUÉS: COLCHONETA_PREMIUM.jpg

                        

                        

                            
ANTES: foto_producto.jpg

                            
DESPUÉS: MANCUERNAS_5KG.jpg

                        

                        

                            
ANTES: product_image.jpg

                            
DESPUÉS: PESAS_RUSAS_10KG.jpg

                        

                    

                

                
                

                    
 Consejos para nombrar imágenes:

                    
    
                        
  • Nombres que funcionan mejor:
    • 
                        
  • CORBAG10K.jpg → PERFECTO (coincide con clave)
    • 
                        
  • colchoneta_premium_100x50.jpg → BUENO (contiene palabras clave)
    • 
                        
  • mancuernas_hexagonal_5kg_par.jpg → BUENO (descripción detallada)
    • 
                        
  • Nombres a evitar:
    • 
                        
  • IMG_1234.jpg → MALO (no hay coincidencia)
    • 
                        
  • DSC_9876.jpg → MALO (no hay coincidencia)
    • 
                        
  • foto.jpg → MUY MALO (muy genérico)
    • 
                    

                

                
                

                    
 Herramientas recomendadas para renombrar:

                    
    
                        
  • Windows: Bulk Rename Utility (gratis)
    • 
                        
  • Mac: NameChanger (gratis)
    • 
                        
  • Online: RenameFilesOnline.com
    • 
                        
  • Multiplataforma: Advanced Renamer
    •
